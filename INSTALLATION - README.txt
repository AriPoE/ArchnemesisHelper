1. Install: https://www.oracle.com/java/technologies/downloads/#jdk17-windows
2. Extract opencv_java455.zip/rar
2.1 Alternativly you can download the dll yourself at https://opencv.org/releases/
3. Put opencv_java455.dll in java JDK path (Default: C:\Program Files\Java\jdk-17.0.2\bin)
4. Double-Click ArchnemesisHelper.jar / ArchnemesisHelper.exe / StartIfNoJRE.cmd
5. ArchnemesisInventory needs to be open and visible and programm needs writing permission in it's rootfolder (for screenshot)
